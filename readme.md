# Verilog Projects

A collection of digital design projects implemented in **Verilog HDL**, covering combinational logic, sequential logic, and digital system design. These projects are intended for learning, experimentation, and FPGA/ASIC design practice.

## Repository Structure

```
VERILOG/
├── async_FIFO/         # Asynchronous FIFO implementation
├── MIPS32/             # 32-bit pipelined MIPS processor
└── vedic_multiplier/   # Vedic multiplier implementations
```

## Projects

### MIPS32

A 32-bit pipelined MIPS processor implementing the core stages of instruction execution, including instruction fetch, decode, execute, memory access, and write-back.

### Vedic Multiplier

Implementation of an 8×8 unsigned Vedic multiplier using the Urdhva-Tiryagbhyam algorithm. The project includes both a hierarchical recursive architecture and a generate-based parameterized architecture.

### Asynchronous FIFO

An asynchronous FIFO for reliable data transfer between independent clock domains, featuring separate read/write clocks and status flag generation.
